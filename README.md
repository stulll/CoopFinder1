# CoopFinder

Проект представляет собой веб-сайт для поиска сокомандников в кооперативных онлайн-играх.

## Описание
Сайт содержит:
- Главную страницу с описанием задачи и преимуществами сервиса.
- Форма регистрации и логина (бэкенд-заготовка).
- Анкету игрока и создание лобби.
- Страницу поиска с фильтрами.
- Место для вставки картинок и базы данных.

## Технологии
- Node.js
- Express
- SQL база данных (через SQL Database Projects extension)
- HTML, CSS, JavaScript

## Настройка базы данных

### SQL база данных (текущий вариант)
Проект использует SQL базу данных через SQL Database Projects extension.

1. Откройте папку `DatabaseProject` в VS Code.
2. Настройте подключение к вашей SQL базе данных (MySQL, PostgreSQL и т.д.).
3. Опубликуйте проект базы данных для создания таблиц.
4. Обновите `.env` файл с настройками подключения:
   ```
   DB_HOST=localhost
   DB_USER=your_username
   DB_PASSWORD=your_password
   DB_NAME=coopfinder
   ```

### Структура базы данных
- `users` — аккаунты пользователей
- `profiles` — анкеты с ролями, играми, платформами
- `lobbies` — объявления о поиске игроков
- `applications` — отклики на лобби
- `reviews` — отзывы и репутация

## Запуск
1. Установите зависимости:
```bash
npm install
```
2. Настройте базу данных (см. выше).
3. Запустите сервер:
```bash
npm start
```
4. Откройте `client/index.html` в браузере или зайдите на `http://localhost:5000`
